css.js

CSS Helpers with Javascript

Use inline css helpers in a fashion way

<!-- Add margin-top of 25px -->
<div class="mt25"></div>
<!-- Add font-size of of 50px -->
<div class="fs50">hello!</div>
<!-- Combine helpers -->
<div class="fs50 mt25 mb0 ml50 br30%">hello!</div>
var observe = true

// use without jquery
var scope = window.document.getElementById('container');
window.cssjs.apply(scope, observe);

// with jquery
var scope = $('#container');
window.cssjs.apply(scope, observe);

Try it

Open console (F12) and write

window.cssjs.apply(document, true);

now start setting helpers in element class attributes
